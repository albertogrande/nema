# Support

Thanks for using Nema. Here's the fastest way to get the help you need.

| I want to… | Go to |
|---|---|
| **Ask a question / discuss an idea** | [GitHub Discussions](../../discussions) |
| **Report a bug** | [Open an issue](../../issues/new/choose) → Bug report |
| **Request a feature** | [Open an issue](../../issues/new/choose) → Feature request |
| **Report a security vulnerability** | **[SECURITY.md](SECURITY.md)** — please do *not* open a public issue |
| **Understand how to contribute** | [CONTRIBUTING.md](CONTRIBUTING.md) |
| **Learn the agent contract** | [CLAUDE.md](CLAUDE.md) / [AGENTS.md](AGENTS.md) |

Nema is in **alpha** and maintained by a small team — please be patient and provide as much
detail as you can (versions, a minimal reproduction, and what you expected). Concrete,
reproducible reports get resolved fastest.
