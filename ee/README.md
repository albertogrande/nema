# `ee/` — Reserved (empty in v0.x)

This directory is **reserved** for a future source-available commercial tier of Forge
(hosting, SSO, governance dashboards, managed/signed attestation, support).

It is **empty in v0.x** and intentionally so. Reserving it now means the open-core boundary can
land later **without relicensing any core code** — everything outside `ee/` is and remains
Apache-2.0.

- Code in `ee/` is **out of scope** for the repository's Apache-2.0 license.
- Code in `ee/` is **out of scope** for the core DCO; do not contribute here.
